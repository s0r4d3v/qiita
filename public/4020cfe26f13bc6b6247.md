初回プッシュのときにしか必要ないコマンドも含まれてるけどもう最初で毎回手間取るのがめんどくさいので全部ひっくるめてやった。
あまり人間をなめるなよ。

```batch:push.bat
@echo off
cd %~dp0

git add -A
git commit -m %1

git config pull.rebase false
git pull --allow-unrelated-histories origin main
git push -u origin main
```

使い方：
引数にcommit message（ダブルクオーテーションで囲む） を書いて実行

例：
```batch
./push.bat [コミットメッセージ]
```
